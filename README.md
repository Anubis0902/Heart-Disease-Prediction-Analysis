# Heart Disease Prediction & Analysis

A comprehensive, end-to-end Machine Learning data science project for analyzing clinical data and predicting the likelihood of heart disease in patients. This project involves Data Preprocessing, Exploratory Data Analysis (EDA), Model Training & Evaluation, and a final interactive Streamlit Application.

## Project Structure

- `01_data_preprocessing.ipynb`: Jupyter Notebook detailing the data cleaning processes (handling missing values, addressing outliers, and feature engineering).
- `02_EDA.ipynb`: Comprehensive Exploratory Data Analysis utilizing visualizations to uncover trends, distributions, and correlations within the heart disease dataset.
- `03_model_training_evaluation_prediction.ipynb`: A step-by-step walkthrough of model selection, hyperparameter tuning, model evaluation (accuracy, precision, recall), and final model extraction.
- `app.py`: A complete Streamlit Dashboard that includes real-time heart disease predictions using our validated best machine learning model, alongside an interactive EDA section containing dynamic charts.
- `best_heart_disease_model.pkl`: The serialized version of the optimally trained ML model.
- `scaler.pkl`: The serialized `StandardScaler` ensuring any input data passed matches the scaled structure expected by the model.
- `heart_disease_data.csv`: The initial raw heart disease dataset.
- `heart_disease_cleaned.csv`: Cleaned and ready-to-use dataset resulting from the preprocessing pipeline.
- `requirements.txt`: Python package dependencies needed to run the project.

## How to Run the Application

1. Make sure Python 3.8+ is installed on your environment.
2. Open a terminal/command prompt in the directory (`e:\DS MINI PROJECT`).
3. (Optional but recommended) activate a virtual environment.
4. Install all the necessary packages via the following command:
   ```bash
   pip install -r requirements.txt
   ```
5. Launch the Streamlit application:
   ```bash
   streamlit run app.py
   ```

## Features of the Streamlit App

- **🩺 Predictive Model (Tab 1)**: Enter a patient's customized physiological parameters (Age, Chest Pain Type, Max Heart Rate, etc.) and instantly predict their heart disease risk using our best-performing generalized model.
- **📊 Exploratory Data Analysis (Tab 2)**: Dynamic charts generated by Plotly covering age distribution, correlation heatmaps, max heart rates relative to disease markers, and various other insights.
